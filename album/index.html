Dial up or Broadband?

The only difference between the two is that the high speed album contains higher quality pictures.

If you are using a modem to dial into an internet service provider like AOL, Earthlink, or Walmart Connect, please use the dial up option below.

If you are connected to the internet via a DSL or Cable Internet provider then I recommend that you use the broadband option below.

Beware: If you are using a dialup internet connection and choose to use the broadband option below you may be waiting a very long time for the album and pictures to load.
