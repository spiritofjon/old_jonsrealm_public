HTML Code Tips

Tables


Tables are one of the most essential features in HTML. Without tables it would be very difficult to get a nice layout.

The format of tables are in general:
<TABLE BORDER="2">
<TR><TD>Input</TD><TD>Input</TD></TR>
<TR><TD>Input</TD><TD>Input</TD></TR>
</TABLE>

The result of this peace of html-code will be:
Input Input
Input Input

  • Rows are defined by the commands <TR></TR>.
  • Columns are defined by the commands <TD></TD>.
  • Tables themselves are opened and closed with <TABLE></TABLE>.

You can put as many rows and columns in the table as you want. The size of the cells in the table will automatically be adjusted to the size of the text inside the cells.

Border:
When you don't specify the BORDER command between the TABLE tags, there will not appear a border between the cells or around the table. When this is wanted add the <BORDER="n"> command to the table tag.

Cell/row spaning:
It's also possible to let a cell span more than one column. With <TD COLSPAN="n"></TD>, you can let the cell span over for example three columns. This is useful for more complicated tables. It's also possible to let a cell span more than one rows. This is done by the <TD ROWSPAN="n"></TD> command.
