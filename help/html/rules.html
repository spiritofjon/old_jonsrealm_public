HTML Code Tips

Horizontal rules

A horizontal rule is made with the <HR> command. This command does not require a closing command. The rules will be centered automatically, but can be aligned by adding the appropriate command. See the examples of different code layouts below.

A standard no frills rule: <HR>

A rule with a width of 100: <HR width=100>

A rule 200 wide and a size of 10: <HR size="10" width="200">

Same rule as above except blue: <HR size="10" width="200" color="#0000ff">
