HTML Code Tips

Paragraphs & Spacing

Paragraph Breaks

<P> starts a new line or paragraph separated by a blank line. To get a simulate carriage return, you'll have to use the <BR> tag.

Additional Spaces

If you want more than one space between two words, you have to use the following tag: &nbsp; each of these tags will insert one space.

No Break

This command means that all the text between the start <NOBR> and end </NOBR> of the command will not have line breaks inserted between them.
