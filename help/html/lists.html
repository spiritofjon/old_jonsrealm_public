HTML Code Tips

Lists

Ordered Lists

Ordered lists are often called numbered lists. In an ordered list <OL> is used to open the list and </OL> to close the list. The code layout is as follows:

<OL>
<LH>Header (optional)</LH>
<LI>First list item</LI>
<LI>Second list item</LI>
<LI>Third list item</LI>
</OL>

This is what an ordered list looks like:

    Shopping List
  1. Apples
  2. Milk
  3. Bread

Ordered lists have five options for numbering. Ordered lists by default use regular numbers. By adding the type="XX" tag to the <OL> command you can change these options. The other four options are as follows:

<OL type="A">

    Capital Letters
  1. Apples
  2. Milk
  3. Bread

<OL type="a">

    Lowercase Letters
  1. Apples
  2. Milk
  3. Bread

<OL type="I">

    Capital Roman Numerals
  1. Apples
  2. Milk
  3. Bread

<OL type="i">

    Lowercase Roman Numerals
  1. Apples
  2. Milk
  3. Bread


Unordered Lists

Unordered lists are often called bulleted lists, because of their use of bullets instead of letters or numbers. Unordered lists use <UL></UL> to open and close the list.

Unordered lists have three bullet options. Unordered lists by default use the round bullet. The three options are as follows:

<UL>

    Default Bullets
  • Apples
  • Milk
  • Bread

<UL type="square">

    Square Bullets
  • Apples
  • Milk
  • Bread

<UL type="circle">

    Circle Bullets
  • Apples
  • Milk
  • Bread

With unordered lists you can also use your own image as a bullet instead of the standard bullets. In order to accomplish this however, you have to use some code from Cascading Style Sheets. I won't get into the details of CSS, however, it is a great coding script that adds alot of function and detail to websites. Which is why I use it heavily in this site.

In order to use your own graphic you need to add this STYLE="list-style-image: url(1234.gif)" tag to the <UL> command. Replace 1234.gif with the location of your graphic file. Here is an example of what this looks like

    My own bullet
  • Apples
  • Milk
  • Bread


Definition Lists

Definition lists differ from both ordered and unordered lists in that this list is divided into two parts: the term and the definition. Definition lists use <DL></DL> to open and close the list. The term is enclosed in the commands <DT></DT>. The definition is enclosed in the <DD></DD> tags. It's also possible to add a list header.

Definition list format:
<DL>
<LH>Header</LH>
<DT>Term</DT><DD>Definition</DD>
<DT>Term</DT><DD>Definition </DD>
</DL>

Definition list example:

In what city was I Born?
I was born in Portland, Oregon.
What year was I born?
I was born in the year 1977.


Nested Lists

<UL>
<LI>You can have a list.</LI>
<OL>
<LI>Inside of another list.</LI>
<OL>
<LI>Inside of again another list.</LI>
</OL>
</OL>
</UL>

Which looks like this:

  • You can have a list.
    1. Inside of another list.
      1. Inside of again another list.
