HTML Code Tips

Links


Links to another file:

To link to another homepage or file, you have to use the command <A HREF="http://www.domain.com/path/file.html"> desription of the file </A>.

To link to another file on the same server, you can use <A HREF="path/file.html"> description of the file </A>.



Links in the same file:

When you want to link to a part in the same file, you have to do two things:

  1. First you have to place a pointer at the place in the file where you want to link to. The pointer looks like <A NAME="Place">.

  2. Secondly, you have to link to that particular spot. That link has to look like <A HREF="#Place"> name/description of that place </A>

You could also link to a part of another page. It can be done in the same way: <A HREF="http://www.domain.com/path/file.html#Place">



Open links in a new window:

This is done by adding TARGET="_BLANK" to the anchor command. It will look like: <A HREF="URL" TARGET="_blank">



E-mail links:

This is done almost the same way as document-links. Just use the tag <A HREF="MAILTO:username@address">text</A>.
