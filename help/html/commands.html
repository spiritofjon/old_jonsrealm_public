HTML Code Tips

Text Formatting Commands

There are way to many tags for me to list, however here are some of the commands used most often. These tags in general will take the form:<command> text </command>

Keep in mind that most HTML commands are not type sensitive, you can use either lower or upper case text, however you must close out these commands or you will not like the results

  • <B> bold </B>
  • <I> itallics </I>
  • <U> underline </U>
  • <BIG> big print </BIG>
  • <SMALL> small print </SMALL>
  • <SUB> subscript </SUB>
  • <SUP> superscript </SUP>
  • <STRIKE> strikeout </STRIKE>
  • <ADDRESS>
    John Doe
    1234 Anywhere St.
    Yourtown, USA 55555
    </ADDRESS>
  • <PRE>
     preformatted text
    </PRE>
